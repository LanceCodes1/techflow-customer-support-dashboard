# TechFlow Customer Support Dashboard

A data analytics and customer support dashboard project built with Python, SQLAlchemy, SQLite, Pandas, Streamlit, and Loveable.

## Overview

This project simulates customer support operations for a fictional company called TechFlow Solutions. The dashboard analyzes customer support call data to identify operational bottlenecks, customer satisfaction trends, and staffing opportunities.

The project was developed as part of The Knowledge House AI Business Solutions track.

---

## Features

- Generated realistic customer support call data using Faker
- Built SQLite database with SQLAlchemy
- Exported datasets using Pandas
- Created interactive Streamlit dashboard
- Built visual analytics dashboards in Loveable
- Developed business recommendations based on support trends

---

## Key Findings

- Average support call duration was approximately 30 minutes
- Customer satisfaction averaged around 3 out of 5
- Billing and account access issues appeared most frequently
- Monday and Friday afternoons showed the highest support demand

---

## Recommended Business Solution

TechFlow should increase staffing during peak support hours and improve billing/account access workflows to reduce customer wait times and improve overall satisfaction.
